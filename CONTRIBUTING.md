# Contributing Guide

Thanks for your interest in improving this project.

## Development Setup

1. Create and activate a virtual environment.
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Keep `raw_sales.xlsx` in the project root before running `sales_cleaner.py`.

## Contribution Rules

- Keep changes focused and small.
- Preserve current script behavior unless the change explicitly targets a bug.
- Add clear commit messages.
- Update `README.md` when introducing user-facing changes.

## Pull Request Checklist

- Code runs successfully locally.
- No accidental generated files are included.
- Documentation is updated when needed.
- PR description explains what changed and why.
